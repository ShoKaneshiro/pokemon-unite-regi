# Pokemon Unite Regi Predictor

Pokemon Uniteのレジ系ポケモン出現パターン予測アプリ

## 機能

- 7:00の組み合わせから4:30と3:00の予測
- 頻度ベースの予測アルゴリズム
- 実際のゲーム結果を記録して予測精度向上
- 過去の記録を可視化

## 使い方

1. 7:00のポケモンを選択
2. 予測ボタンをクリック
3. 4:30と3:00の予測結果を確認
4. 実際のゲーム結果を記録フォームから追加
